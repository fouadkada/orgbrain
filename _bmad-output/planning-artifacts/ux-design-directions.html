The v1 API was deprecated in Q1 2024 primarily due to authentication model limitations — it used static API keys with no per-scope permissions, which became a compliance risk as the org moved toward SOC 2 preparation. Rafael's platform team proposed the change in #eng-platform; the decision was ratified in the February architecture review.

The v2 API replaced it with OAuth 2.0 token-based auth, with the migration window closing June 2024. All internal services completed migration by May.
